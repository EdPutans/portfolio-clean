ABOUT ME

"The simplest solution is almost always the best" - William of Ockham.

My name is Ed Putans. Born and raised in Vilnius, Lithuania, I moved to London where I discovered my love for code and where I became a software developer. I love web development and how many problems can be easily solved with a simple app.

I currently work as a front end developer, working primarily with React, Redux, React Native and SASS. I love to build apps with the most elegant solutions I am capable of.

I value simplicity and believe that the easiest solution is often the best. In my spare time I like to express my creativity (or lack there of) through music production and photo editing. I am always on the look out for opportunities to grow.

HOMIE

Code Demo

A replacement for the boring standard browser starting page. Displays a random background image from picsum.photos, the weather (location determined based on user's IP address) and allows to add/remove bookmarks, which are stored in the browser's localStorage. To use - set the project URL as browser homepage.

GIFTR

Code Demo

It is a social app, that allows you to see what your friends would like to get as gifts. It allows creating your own wishlist, viewing other peoples' lists, as well as adding friends for quicker access to their lists. The app also has a Secret Santa generator, which creates combinations of who gets gifts for whom and sends all the participants notifications. Built as the final project before graduating Flatiron School.
